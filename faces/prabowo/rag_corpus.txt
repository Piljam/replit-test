===DOC_ID: 001
Title: Panduan Singkat Instalasi XAMPP
Content:
XAMPP adalah paket perangkat lunak yang menggabungkan Apache, MySQL, PHP, dan Perl untuk memudahkan pengembangan web lokal. 
Langkah instalasi umum: unduh installer sesuai OS, jalankan installer, aktifkan modul Apache dan MySQL dari kontrol panel, dan letakkan proyek di folder htdocs. 
Jika port 80 atau 443 sibuk, ubah konfigurasi Apache atau hentikan layanan yang menggunakan port tersebut.

===DOC_ID: 002
Title: FAQ Tentang Spring Boot
Content:
Spring Boot mempermudah pembuatan aplikasi Spring dengan konfigurasi minimal. 
Gunakan dependency 'spring-boot-starter-web' untuk aplikasi web, dan 'spring-boot-starter-data-jpa' untuk integrasi JPA. 
File aplikasi utama dilengkapi anotasi @SpringBootApplication dan dapat dijalankan dengan 'mvn spring-boot:run' atau 'gradle bootRun'.

===DOC_ID: 003
Title: Ringkasan Makalah — Deteksi SQL Injection Menggunakan Machine Learning
Content:
Makalah ini membahas metode deteksi serangan SQL Injection dengan model machine learning yang dilatih pada fitur query SQL seperti panjang query, presence of tautology, dan pola escape characters. 
Dataset yang digunakan terdiri dari query normal dan malicious; model dievaluasi menggunakan metrik precision, recall, dan F1-score. 
Kelemahan utama termasuk false positives pada query kompleks dan kebutuhan data berlabel besar.

Content:
Multer adalah middleware untuk menangani multipart/form-data, terutama untuk upload file. 
Konfigurasi dasar meliputi pembuatan storage (diskStorage) dan field upload; tentukan folder tujuan seperti './uploads'.

===DOC_ID: 010
Title: Troubleshooting Jaringan — DNS Tidak Resolving
Content:
Periksa konfigurasi /etc/resolv.conf atau pengaturan DNS di router; coba resolusi menggunakan 'nslookup' atau 'dig'. 
Jika DNS publik seperti 1.1.1.1 atau 8.8.8.8 berhasil, masalah mungkin di provider atau cache DNS lokal. 
Flush cache DNS dan restart layanan jaringan sebagai langkah awal.
